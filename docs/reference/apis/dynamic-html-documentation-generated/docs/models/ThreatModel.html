ThreatModel

  • id : UUID
    Unique identifier for the threat model (UUID)
  • created_at : Date
    Creation timestamp (RFC3339)
  • modified_at : Date
    Last modification timestamp (RFC3339)
  • created_by : String
    User name, email or identifier of the creator of the threat model
  • documents : List
    List of documents related to the threat model
  • repositories : List
    List of source code repositories related to the threat model
  • diagrams : List
    List of diagram objects associated with this threat model
  • threats : List
    List of threats within the threat model
  • notes : List
    List of notes associated with the threat model
  • assets : List
    List of assets associated with the threat model
  • status_updated : Date
    Timestamp when the status field was last modified (RFC3339). Automatically updated by the server when status changes.
  • name : String
    Name of the threat model
  • description : String
    Description of the threat model
  • owner : String
    Email address of the current owner
  • threat_model_framework : String
    The framework used for this threat model
  • authorization : List
    List of users and their roles for this threat model
  • metadata : List
    Key-value pairs for additional threat model metadata
  • issue_uri : String
    URL to an issue in an issue tracking system for this threat model
  • status : List
    Status of the threat model in the organization's threat modeling or SDLC process. Examples: \"Not started\", \"In progress\", \"Review\", \"Approved\", \"Closed\"
