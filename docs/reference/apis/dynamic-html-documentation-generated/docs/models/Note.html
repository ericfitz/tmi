Note

  • id : UUID
    Unique identifier for the note
  • metadata : List
    Optional metadata key-value pairs
  • created_at : Date
    Creation timestamp (RFC3339)
  • modified_at : Date
    Last modification timestamp (RFC3339)
  • name : String
    Note name
  • content : String
    Note content in markdown format
  • description : String
    Description of note purpose or context
