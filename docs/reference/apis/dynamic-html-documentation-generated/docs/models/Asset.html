Asset

  • id : UUID
    Unique identifier for the asset
  • metadata : List
    Optional metadata key-value pairs
  • created_at : Date
    Creation timestamp (RFC3339)
  • modified_at : Date
    Last modification timestamp (RFC3339)
  • name : String
    Asset name
  • description : String
    Description of the asset
  • type : String
    Type of asset
    Enum:
    data
    hardware
    software
    infrastructure
    service
    personnel
  • criticality : String
    Criticality level of the asset
  • classification : List
    Classification tags for the asset
  • sensitivity : List
    Sensitivity labels for the asset
