DfdDiagram

  • type : String
    DFD diagram type with version
    Enum:
    DFD-1.0.0
  • cells : List
    List of diagram cells (nodes and edges) following X6 structure
  • id : UUID
    Unique identifier for the diagram (UUID)
  • name : String
    Name of the diagram
  • created_at : Date
    Creation timestamp (ISO3339)
  • modified_at : Date
    Last modification timestamp (ISO3339)
  • metadata : List
    Key-value pairs for additional diagram metadata
  • update_vector : Long
    Server-managed monotonic version counter, incremented on each diagram update
  • image : BaseDiagram_image
  • description : String
    Optional description of the diagram
