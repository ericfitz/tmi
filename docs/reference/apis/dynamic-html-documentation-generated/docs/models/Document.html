Document

  • id : UUID
    Unique identifier for the document
  • metadata : List
    Optional metadata key-value pairs
  • created_at : Date
    Creation timestamp (RFC3339)
  • modified_at : Date
    Last modification timestamp (RFC3339)
  • name : String
    Document name
  • description : String
    Description of document purpose or content
  • uri : String
    URL location of the document
